# ホームページ チュートリアル

これは最初のチュートリアルで作成したモダンなホームページです。

## 特徴

- レスポンシブデザイン（モバイル対応）
- モダンで美しいUI
- スムーズスクロール
- アニメーション効果
- お問い合わせフォーム

## ファイル構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `script.js` - JavaScriptファイル

## 使い方

1. `index.html`をブラウザで開く
2. または、ローカルサーバーを起動して表示する

```bash
# Python 3を使用する場合
python3 -m http.server 8000

# Node.jsを使用する場合（http-serverが必要）
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

## セクション

- **ホーム**: ヒーローセクション
- **について**: サイトの説明
- **サービス**: 提供サービスの紹介
- **お問い合わせ**: コンタクトフォーム

## カスタマイズ

`styles.css`の`:root`セクションで色やスタイルをカスタマイズできます。
